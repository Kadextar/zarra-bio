# Zarra Resort — link-in-bio

Одностраничный link-in-bio лендинг горного резорта **Zarra Resort** (Самарканд).
Тёмная тема с золотом, анимированные золотые линии на canvas, переключатель языков RU / UZ / EN.
Родственный лендингу [Safo Restaurant](https://safo-bio.netlify.app) — бренды одной семьи.

## Стек

Чистый HTML5 + CSS + Vanilla JS, без сборщиков и внешних библиотек. Разметка в `index.html`,
картинки в `assets/` (шапка, логотип, og-превью).

## Деплой на Netlify

Сайт привязан к этому репозиторию. Любой `git push` в `main` пересобирает сайт автоматически.
Настройки сборки пустые (статический сайт): **Build command** — пусто, **Publish directory** — `.`
